Data

Datasets are stored for free on the DANDI archive. Here we provide a curated list of DANDIsets provided by various labs with basic descriptions of the contents of each dataset and links to the DANDI page where they are housed. The DANDI page should contain links to the files in the dataset, relevant metadata for the experiments, and information about the contents of the NWB files.

If using Python: Install pynwb in python environment and import NWBHDF5IO from pynwb into python file to run code to read NWB files. See pynwb documentation for more information.

If using Matlab: Download MatNWB and generate API to interface with NWB files in Matlab. See MatNWB documentation for more information.

See page on adding your own dataset to learn how to add your own data to this list.
