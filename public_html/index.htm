Welcome to WormID.org!

A comprehensive resource for the C. elegans neuroscience community to facilitate data standardization, storage, and sharing as well as provide tools for data conversion and pipelines for analysis of C. elegans whole-brain imaging.

About

We aim to simplify the process of developing new approaches for labeling C. elegans neurons in volumetric images by providing a resource for tools and code for facilitating neuron ID and related data analysis tasks as well as a consolidated database of various imaging datasets from labs around the world. We hope these resources will expedite the development of new algorithms for automatic identification of C. elegans neurons and enable new discoveries.

We provide links to existing databases of labeled NeuroPAL images and GCaMP time series stored in the Neurodata without borders (NWB) format on the DANDI Archive as well as code for pre-processing, neuron segmentation, curation, and identification of novel NeuroPAL images.

Data

We introduce a novel extension to the Neurodata Without Borders (NWB) HDF5-based file format to support data associated with volumetric calcium imaging in C. elegans. We believe NWB will be an extremeley useful tool for the worm neuroscience community to standardize file formats and data storage solutions in order to facilitate data sharing and collaborations.

The data page contains links to datasets stored on the DANDI archive and information about how to download and use NWB files stored on the DANDI archive.

See our page on adding your own dataset to learn how to convert your own data to NWB, upload it to DANDI, and add it here to contribute to this community resource.

Code

The code page contains descriptions and links to relevant code for reading, processing, and analyzing NeuroPAL images and GCaMP time series.

We provide a few example pipelines for segmentation, curation, and quantification of neurons from raw data. This page will be updated as these pipelines continue to develop.

We hope to establish a community framework for C. elegans whole-brain imaging analysis pipelines that carries data through the whole process starting from raw NeuroPAL structural images and calcium imaging data to segmented neuron locations, cell identity labels, and quantified calcium activity for each cell.

Adding your own data

The 'add your data' page contains comprehensive tools and information for converting your own calcium imaging data into the NWB format, uploading the data to DANDI, and adding it here to contribute to this community resource.

We hope to continue to grow this resource of existing datasets to facilitate new research and the development of new tools by providing datasets to be used for training and testing of both new and existing approaches.
